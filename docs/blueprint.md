# **App Name**: Bhattarai Deals

## Core Features:

- Product Showcase: Display iPhones, AirPods, and accessories in a minimal grid layout with large photos, clean titles, prices, and condition badges.
- Condition Grading: Implement an official-feeling condition grading system (Brand New, Like New, Certified Used) with detailed descriptions of each category.
- Product Detail Pages: Showcase individual items in a full-page view that makes use of smooth zoom functionality.
- Smart Comparison: Allow users to compare two devices side-by-side, highlighting specs, camera differences, battery life, and price/condition variations. If the characteristics of either one of the comparison models would affect the user, the AI tool will include that information in its comparison.
- Curated Deals Hub: Feature power banks, smartwatches, cases, headphones, and chargers from other brands while maintaining a premium layout and feel.
- Confidence Zone: Build user trust with a dedicated section for IMEI verification, warranty policies, return policies, and authenticity certificates.
- Inventory Checker: Show real-time availability statuses like 'Available now' or 'Coming soon' for each product.

## Style Guidelines:

- Primary background: #F0FAFF. Almost white, used minimal. 98% neutral, 2% blue tint.
- Primary: #B0E2FF. Used for CTAs, active states, highlights.
- Accent: #D0B0FF. Used for secondary buttons, badges, tiny UI accents.
- Font: SF Pro Display (Headings) and SF Pro Text (Body). Fallback to Inter.
- Buttons: Primary - White text, Background: #B0E2FF, Radius: 12px, Subtle shadow. Secondary - Text: #222, Border: 1px solid #D0B0FF, Background: transparent.
- Cards: Background: white, Radius: 20px, Shadow: ultra-soft 0/0/20 RGBA(0,0,0,0.04), Minimal text, Massive product image dominates.
- Navigation Bar: Fully white, Slight translucency, Minimal blurred background on scroll, Thin underline / divider: #E8F7FF.
- Hero Section: Giant iPhone photo, Heading: bold, large, centered, Sub-heading: ultra minimal, Buttons: soft blue (#B0E2FF) primary, pale purple secondary, Background: white.
- Category Sections: Full-width section for iPhones and accessories. Each with massive imagery + very little text.
- Comparison Tables: Background: #FAFAFB, Border: 1px #F0F0F2, Active column border: #B0E2FF.
- Micro-interactions: Fade-in animation (0.2–0.3s), Smooth transitions on hover, Magnetic button effect optional, Soft parallax on hero images.
- Brand Tone: Clean, Direct, Simple, Confident. Example: “Premium Devices. Simple Prices.”
